# Multivendor E-commerce Web App

## 🌟 Overview
Full-stack Multivendor E-commerce platform built with **Django, Django REST Framework, React, PostgreSQL, Stripe API, and AWS S3**.  
Designed for **scalable architecture**, secure user & vendor workflows, and real-world functionality.

**Live Demo:** [Advance Multivendor Web App](https://advance-multivendor-webapp.netlify.app/)  
**Video Demo:** [YouTube](https://youtu.be/9ntYVsddJ0k)

---

## 🛒 Key Features

**User Side:**
- Authentication (HTTP-only cookies & email verification)
- Secure Stripe payments
- Orders, wishlist, reviews
- Real-time notifications

**Vendor Side:**
- Product & order management
- Earnings analytics & sales tracking
- Coupon & review system
- Notifications

**System Highlights:**
- Scalable multivendor architecture
- Secure REST API (Django REST Framework)
- Fully custom React frontend

---

## 🛠 Tech Stack
- **Backend:** Python, Django, Django REST Framework
- **Frontend:** React, JavaScript, HTML, CSS
- **Database:** PostgreSQL
- **Deployment:** Railway (backend), Netlify (frontend), AWS S3 (media & static files)
- **Payments:** Stripe API
